Alur Kerja 5 Tahap Pipeline SODS Produksi

1

COLD RUN — Instrumentasi Non-Invasif & Taint Analysis

Selective Specialization

Aplikasi biner tertutup (Electron/WASM/Python) dijalankan pertama kali. Pengeksekusi memantau dan memilah fungsi berstatus I/O secara otomatis.

Taint Analysis (Mozilla rr) Fungsi PURE ditandai sebagai kandidat spesialisasi; Fungsi IMPURE / I/O dialirkan langsung ke Host OS agar aman.
PMU Sampling & PEP 669 Hooks Interupsi timer prosesor tiap 10ms + Python 3.12+ sys.monitoring memetakan Hot Paths dengan overhead <1% CPU.
▼
2

SPECIALIZER — Bangkitkan Jalur Silikon Cepat & Guards

Zero Modification Bounds

Begitu profil pemanggilan stabil (>50 eksekusi), sistem memancarkan Assembly tingkat silikon dengan injeksi pengaman Guard.

eBPF + DynamoRIO Kernel Probes Mencegat proses yang sedang berjalan dan menyuntikkan Probe observasi tanpa memodifikasi 1 bit pun berkas aslinya.
Polymorphic Inline Caches (PIC) Membangkitkan eksekusi komputasi cepat dengan Thread-Local Guards untuk kombinasi masukan teramati (maksimal 3 tipe).
▼
3

EQUIVALENCE VERIFICATION — Validasi Empiris Terikat Domain

Workaround Teorema Rice

Bandingkan hasil kalkulasi Generic vs Spesialisasi pada domain masukan empiris. Mengakali ketidakterputusan dengan domain terbatas.

Equivalence Verifier Core Jika semua keluaran identik, spesialisasi LULUS. WASI/Seccomp membekukan emisi I/O ganda di peramban pengguna.
Tier-Lowering Protection Bila rasio kegagalan Guard pada masukan acak melampaui 30%, bakar fast path secara permanen kembali ke mode Generic.
▼
4

💾 PERSISTENT COOKIE ATTESTATION BOUNDARY

Metadata profil + spesialisasi silikon + SHA-256 Hash biner aplikasi diserialisasi dan disimpan secara fisik ke media penyimpan OS.

.sods/profile.json Dilindungi oleh Ed25519 HMAC · chmod 400 read-only · Runtime Behavioral Fingerprint Attestation ✓
▼
5

WARM RUN — Eksekusi Melompat 4.5× – 7.14× Lebih Cepat

Ultra High Throughput

Saat aplikasi dibuka kembali: Cookie disedot, sidik jari divalidasi, jalur komputasi terspesialisasi langsung dieksekusi.

Timing Noise Randomization Distorsi pewaktu virtual acak tingkat mikro menumpumpukan teknik serangan Sandbox Evasion Timing Attacks dari malware.
OSR Deoptimizer Core Jika masukan tak terduga melanggar asumsi Guard, eksekusi dievakuasi instan ke Interpreter lambat. Hasil tetap 100% BENAR.

Peta Mitigasi 5 Rintangan Eksternalitas Produksi (Bab 5.6)

H1
Stateful Memory & I/O Side Effects
Selective Spec · Taint Analysis · Mozilla rr

Mozilla rr merekam eksekusi deterministik secara utuh dan me-replay komputasi murni tanpa duplikasi efek samping I/O yang berbahaya.

H2
Jurang FFI & Biner Tertutup V8
eBPF Kernel Probes · DynamoRIO · Intel PIN

eBPF menyuntikkan kprobes/uprobes ke proses berjalan tanpa merubah 1 bit pun biner asli. DynamoRIO menulis ulang Assembly murni *on-the-fly*.

H3
Paradoks Overhead Cold Start
Hardware PMU Sampling · Linux perf sampling

Menyingkirkan *comprehensive tracing*. PMU prosesor membangkitkan interupsi tiap 10ms untuk memetakan Hot Paths tingkat statistik (<1% CPU overhead).

H4
Async Loops & Multithreading
Thread-Local Guards · Conservative Global Deopt

Thread-Local Guards memastikan deopt terjadi mandiri per-thread. Untuk `async/await`: SODS bersikap konservatif mengunci jalur demi kebenaran mutlak.

H5
Keamanan Host & Obfuscated Binaries Bounds
Timing Noise Randomization · Ed25519 Cryptographic Signatures · Runtime Attestation

Distorsi pewaktu virtual acak (*Timing Noise Randomization*) menumpumpukan serangan deteksi sandbox. *Behavioral Attestation* memvalidasi sidik jari di tiap eksekusi Warm Run.

Batas Desain Industri (*Sober Boundary*): SODS secara sadar tidak bekerja pada biner yang sengaja diobfuscasi atau dienkripsi oleh pengembangnya, selaras dengan filosofi kompilator AOT tingkat produksi (GCC/Clang).
